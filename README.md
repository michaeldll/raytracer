# Baby Raycaster

This is a barebones raycaster in C++.

## Reference
https://raytracing.github.io/books/RayTracingInOneWeekend.html#overview.

## Instructions

Build the binary:
`make`

Render an image:
`make render`

Build with debugging flags:
`make build-dev`

Delete artefacts:
`clear`

## Dependencies
- C++14 compiler (e.g., clang++)
- Bash shell (for running scripts)
